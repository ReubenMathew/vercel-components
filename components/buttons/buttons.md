# Vercel Buttons

## Usage

```html
    <Button type="<Type>" mod="<Modifier>">Action</Button>
    <SmallButton type="<Type>" mod="<Modifier>">Action</SmallButton>
```
#### Example
```html
    <Button type="warning" mod="shadow">Action</Button>
    <SmallButton type="warning" mod="shadow">Action</SmallButton>
```

### Types
- primary
- secondary
- success
- error
- warning

### Modifiers
- shadow
